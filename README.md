#  Java Design Patterns

## Creational 创造型

- Singleton 单例模式
- Prototype 原型模式
- Builder 构造器模式
- Factory Method 工厂方法
- Abstract Factory 抽象工厂

## Structural 结构型

- Adapter 适配器模式
- Bridge 桥接模式
- Composite 组合模式
- Decorater 装饰器模式
- Proxy 代理模式
- Facade 外观模式
- Flyweight 享元模式

## Behavioral 行为型

- Observer 观察者模式
- Strategy 策略模式
- Template Method 模板方法
- Chain 责任链模式
- Command 命令模式
- Interpreter 解释器模式
- Iterator 迭代器模式
- Mediator 中介者模式
- Memento 备忘录模式
- State 状态模式
- Visitor 访问者模式
